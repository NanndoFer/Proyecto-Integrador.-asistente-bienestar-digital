# Proyecto-Integrador.-asistente-bienestar-digital
Asistente de consola en Python para la gestión de la salud digital y autorregulación.


## Datos del Proyecto
Integrantes: Fernando Mateo Grijalva Zavala
Asignatura:Proyecto Integrador - Ingeniería de Software
Fecha: Junio 2026

Objetivo del Sistema
Desarrollar una aplicación de consola en Python que actúe como un asistente interactivo para concientizar a los usuarios sobre su consumo diario de pantallas y redes sociales, proveyendo herramientas lógicas de autorregulación y mitigación de la fatiga cognitiva.

Descripción de Funcionalidades
Módulo de Diagnóstico: Evalúa las horas de uso diario mediante estructuras condicionales para determinar el nivel de riesgo del usuario (Bajo, Moderado o Crítico).
Temporizador de Enfoque (Método Pomodoro): Cuenta regresiva automatizada de 25 minutos para fomentar bloques de trabajo libre de distracciones digitales.
Desafíos de Desconexión: Generación aleatoria de retos y actividades analógicas fuera de las pantallas para romper ciclos de hiperestimulación.
